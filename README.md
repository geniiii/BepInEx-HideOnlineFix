# BepInEx-HideOnlineFix
Hides `OnlineFix64.dll` from GetModuleHandleA/W since **some** mods like to silently crash if they find it.

See the accompanying blog post: https://geni.site/blog/anti-anti-piracy-and-hiding-from-getmodulehandle/

## Usage
Download the [latest release](https://github.com/geniiii/BepInEx-HideOnlineFix/releases/latest) and follow the instructions on the Releases page.
